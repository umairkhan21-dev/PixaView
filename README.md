# 🚀 PixaView

PixaView is a responsive testing tool that allows developers to preview how their websites look across different screen sizes and devices in real-time.

## ✨ Features

- 📱 Test responsiveness for multiple screen sizes
- ⚡ Real-time preview
- 🎯 Simple and clean UI
- 🌐 Works with any website URL

👉 https://pixaview.dev

## 🛠 Tech Stack

- React.js
- Vite
- CSS
