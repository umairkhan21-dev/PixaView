# 🚀 PixaView

PixaView is a responsive testing tool that allows developers to preview how their websites look across different screen sizes and devices in real-time.

## ✨ Features

- 📱 Test responsiveness for multiple screen sizes
- ⚡ Real-time preview
- 🎯 Simple and clean UI
- 🌐 Works with any website URL

🧩 Chrome Extension

PixaView is also available as a Chrome Extension for instant responsive testing directly from your browser.

🚀 Features
- One-click access from browser toolbar
- Automatically detects current website
- Instant responsive preview without switching tabs
